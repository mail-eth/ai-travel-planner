# AI Travel Planner

Smart travel itinerary generator using Xiaomi MiMo based on interests and budget

## 🎯 Features

- Powered by Xiaomi MiMo API
- Simple and intuitive interface
- Real-time processing

## 🚀 Quick Start

```bash
git clone https://github.com/mail-eth/ai-travel-planner.git
cd ai-travel-planner
npm install
cp .env.example .env
# Add your MIMO_API_KEY
npm start
```

## 📊 Use Case

This project demonstrates practical AI integration using Xiaomi MiMo API for real-world applications.

## 🛠️ Tech Stack

- Xiaomi MiMo API
- Node.js
- Express
- Simple HTML/CSS frontend
